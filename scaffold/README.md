# scaffold/

이 디렉터리는 [LPFchan/repo-template](https://github.com/LPFchan/repo-template)의 **`scaffold/`를 그대로 복사**해 둔 사본입니다. (이 저장소는 상류를 주기적으로 추적하지 않으므로 **`upstream-intake/`** 와 **`skills/upstream-intake/`** 는 넣지 않았어요.)

## 무엇에 쓰나요

- **새 저장소**에 같은 운영 방식을 옮길 때: 이 폴더 내용을 그 저장소 **루트**로 복사하면 [recreate-prompt.md](../recreate-prompt.md) 절차와 맞아요.
- **kkamppak 제품 진실**은 여기가 아니라 **저장소 루트**의 `REPO.md`, `SPEC.md`, `STATUS.md` 등을 봐요. `scaffold/SPEC.md` 같은 파일은 **업스트림 플레이스홀더**예요.

## 갱신

의도적으로 업스트림과 맞출 때는 repo-template를 clone한 뒤 `scaffold/`를 다시 채우면 돼요.
