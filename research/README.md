# research/

재사용할 만한 조사·근거·대안 비교를 짧은 메모로 남깁니다.

## 이 디렉터리에 넣는 것

- `pmset` 동작, macOS 버전별 차이, 보안 트레이드오프 등 **참고용** 정리
- 파일명 예: `RSH-YYYYMMDD-NNN-short-title.md`

## 넣지 않는 것

- 실행 로그 전문 (→ `records/agent-worklogs/`)
- 확정된 제품 결정만 (→ `records/decisions/`)

새 메모는 상단에 다음을 넣습니다.

- `Opened: YYYY-MM-DD HH-mm-ss KST`
- `Recorded by agent: <id>`
